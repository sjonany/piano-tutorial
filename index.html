Arpeggio tutorial

This is a short tutorial on how you can create delicious arpeggio covers for your favorite songs. I originally wrote this for my brother and he likes anime music, so I'm using that as the recurring song :D

Here's the end result:

Level 1. 1-5-1

A basic starting pattern is to play the first (also known as root) and fifth note of the scale of the chord you are playing. For example, for the chord C, the corresponding scale is C major, and the first note is C, third note is E, and fifth note is G.

Sheet

Take a listen here. Note how the left hand only plays 3 quarter notes then pauses for each chord. Do you see that all the left hand is doing is just the 1-5-1 pattern?

Let’s practice!

The lessons build up on top of each other, so it's important to master each before going to the next. Here are some suggested practice goals:

1. [Left] Practice just the left hand in isolation. Make sure you know the 1's and 5's of each chord.

2. [Left and right] Practice with the right hand. Pay attention to which left and right hand notes have to be played at the same time.

3. [In tempo] Practice against the musescore recording, and make sure you can follow the same tempo without any mistakes. In the "Sheet" section, the musescore widget allows looping. Set "end measure" to 4.

4. [Another song] Repeat the above to another song. Pick one song that you know the chords of. If you don't, you can use ultimate guitar and see if you can play the song using this pattern.

Level 2. Continuous 151

We can make our playing be more continuous by also descending after we reach the highest note. The pattern is now 1515.

Level 3. Twice as fast

Let's play twice as many notes. If we just use 1 and 5, we will have to span a lot of octaves, so let's also add the third of each scale. We'll use the pattern 15135131. Note how there are 8 notes to fill each bar.

Sheet

Take a listen here. Doesn't that sound much cooler? Do you also see that for each chord, they all follow the same 15135131 pattern?

Let’s practice!

WARNING: There is a huge jump in difficulty. If this is your first time, I recommend starting with a slow tempo (e.g. 30 BPM, so you play 1 note per second), and each day increasing by 10 BPM.

1. [Left] Practice just the left hand in isolation. Make sure you can play the notes continuously.

2. [Left and right] Practice with the right hand. Pay attention to which left and right hand notes have to be played at the same time.

3. [In tempo] Practice against the musescore recording, and make sure you can follow the same tempo without any mistakes.

4. [Another song] Repeat the above to another song.
